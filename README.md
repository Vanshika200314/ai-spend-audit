# AI Spend Audit

A high-fidelity B2B SaaS lead-generation utility built for Credex to help startups identify software duplication, redundant license seats, and plan over-commitments. Users receive an instant on-screen audit, submit their details to unlock a personalized AI summary, and get a public shareable URL with Open Graph preview tags.
